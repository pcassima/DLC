Requirements

Before we start we need to go over some requirements.
Don't get discouraged!
This is not meant or intended to be an expensive project. Making your own computer is a really fun and interesting project, for everyone. I want to make and keep this project as accessible to as many people as possible.

prerequisites

The only real requirements that you need to have before starting this project are:

  • A lot of enthusiasm
  • An eagerness to learn
  • A healthy dose of curiosity
  • Free time you can dedicate to this project.
  • A place you can work (and maybe leave a mess)
As long as you stay motivated and find some time you can dedicate to this project, everything will be fine!

A basic understanding of electronics and digital systems is useful but not necessary. I will go through this project step by step from the very beginning. Even if you have absolutely no prior knowledge in electronics, logic gates or digital systems, you will still be able to follow along.
I will try my best to explain everything and provide all the required resources.

Materials

As far as materials that are needed, this is also very limited.
A basic multimeter, breadboards and wire are some essentials.
The rest of required materials, such as IC's, will be added to each relevant section.

In fact this entire project can be made in a logic simulator, such as logisim.
This is an excellent tool for simulating digital schematics. However development stopped in 2013.
A swiss academy developed their own digital simulator, based on logisim. They have released this on github logisim-evolution.
