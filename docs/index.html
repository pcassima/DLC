The Discrete Logic Computer

Introduction

After watching many projects by others (on the internet) and an ever growing interest and desire to know how computers work. I have finally decided to make my very own "breadboard" style computer.

This website will detail my adventures in trying to make this computer. I hope it will inspire you to make your own computer.

If you make your own computer send me a message with a link to yours and I will definitely check it out! Get in touch!

All documentation and design files for this project can be found on the relevant GitHub repository. Feel free to help out on the repository, make your own computer and give feedback.

The entire build process will also be documented on YouTube. You can follow along and use them as a reference for building your own computer. These pages are a transcribed version of the videos. Although they will contain some more detailed information.
